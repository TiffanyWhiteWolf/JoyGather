## 本地运行

### 环境要求

- **Java** 1.8+
- **Maven** 3.6+
- **Node.js** 18+
- **MySQL** 5.7+（需提前创建数据库 `quju_dev`）

### 1. 后端启动

```bash
cd backend

# 修改数据库密码和 AI 配置
# 编辑 src/main/resources/application-local.yml
# 填写 spring.datasource.password 和 quju.ai.api-key

# 启动（首次启动 Flyway 会自动建表）
mvn spring-boot:run -Dspring-boot.run.profiles=local
```

后端启动后运行在 `http://localhost:8080/api`。

### 2. 前端启动

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

前端启动后运行在 `http://localhost:5173`。

### 3. 开始使用

1. 打开 `http://localhost:5173`
2. 注册新账号（邮箱激活页面提供本地激活按钮）
3. 创建活动、报名参与
4. 管理员登录：点击"管理员登录"入口，使用预创建账号登录后台
