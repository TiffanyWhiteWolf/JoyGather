# JoyGather 微信小程序演示版

这是课程展示用的最快可演示版本。小程序不单独建用户系统，直接复用网站后端：

- API：`https://joygather.kyhome.me/api`
- 登录：使用网站邮箱账号和密码
- 同步：Token 调用同一套 Spring Boot REST API，数据写入同一 MySQL

## 打开方式

1. 打开微信开发者工具：`E:\wx\微信web开发者工具\微信开发者工具.exe`
2. 导入项目目录：`D:\桌面\大三下\小学期\JoyGather\miniprogram`
3. 如果要使用自己的个人版小程序账号，把 `project.config.json` 里的 `appid` 从 `touristappid` 改成你的 AppID。
4. 课程演示时，如果微信后台还没配置合法域名，在开发者工具右上角“详情 -> 本地设置”勾选“不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书”。

## 真机预览域名报错

如果手机扫码“预览”时报 `request:fail url not in domain list`：

1. 确认微信公众平台配置的是当前项目的 AppID。
2. 在“开发管理 -> 开发设置 -> 服务器域名”的 `request 合法域名` 填 `https://joygather.kyhome.me`，不要填 `/api`，不要填 IP。
3. 保存时按微信公众平台要求完成管理员扫码确认。
4. 回到微信开发者工具，打开“详情 -> 项目配置”，点击刷新域名信息，重新编译后重新预览。
5. 如果仍然报错，打开调试器 Console，查看 `[JoyGather API]` 后面的实际 URL。当前代码只应请求 `https://joygather.kyhome.me/api/...`。

“真机调试”能跑但“预览”不能跑，通常说明代码没问题，是预览包走了微信后台域名白名单校验。

## 演示路径

1. 首页查看后端活动列表。
2. 进入活动详情。
3. 使用网站账号登录，例如 `demo@quju.cn / 12345678`。
4. 报名或取消报名。
5. 切到“我的”查看报名状态。
6. 编辑个人资料，再到网站个人中心确认同步。
